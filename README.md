# MOUSTACHE

> _"That's a woman in a turquoise tank top and the game is still asking you to choose Tom Selleck or Burt Reynolds."_

**Can you tell Tom Selleck from Burt Reynolds?**
_...Or are they the same person?_

A web-based investigation disguised as a celebrity quiz. 8 rounds of photo identification, followed by a descent into a conspiracy rabbit hole you can't climb out of.

## Play

Open `index.html` in any browser. No build tools, no dependencies, no frameworks.

## What happens

1. You take a quiz
2. You get your score
3. You are asked a question you can't answer
4. You open the file
5. Your truth rating never goes down

## Features

- 14 pieces of evidence, all based on real facts
- Moustache forensics (99.7% match)
- 5 defense layers that absorb your skepticism
- A dossier the game writes about _you_
- VHS decay that gets worse the more you investigate
- A truth rating system with no ceiling and no floor

## Tech

Vanilla HTML/CSS/JS. Web Audio API synth. Zero dependencies. The `--corruption` CSS custom property drives all visual decay from 0 to 1.
